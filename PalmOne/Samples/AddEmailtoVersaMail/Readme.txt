---------------------------------------------------------------------------
Name

  AddEmailtoVersaMail

---------------------------------------------------------------------------
Description

  This sample application shows how to programmatically add an email to 
  VersaMail. It also demonstrates how to send a memo over Bluetooth.

---------------------------------------------------------------------------
Builds With

  CodeWarrior
  Gcc

---------------------------------------------------------------------------
Devices

  All Palm devices with VersaMail

---------------------------------------------------------------------------
Requirements

  Versamail needs to be set up for any type of account

---------------------------------------------------------------------------
Libraries Used

  VersaMail Library

---------------------------------------------------------------------------
How to Run

  1. Launch 'Add EmailCW9' 
  2. Tap on 'Exchange Mgr' button 
     a. Select 'Bluetooth' and choose another Bluetooth enabled device
        - The selected BT device will receive a Testing Memo.  
     b. Select "Email"
        - VersaMail will launch in compose view  
  3. Select 'Helper API' or 'UI Launch VM'
   - VersaMail will launch in 'compose' form with fields filled in
  4. Select 'Sub Launch VM'
   - Not supported on Treo 650 and later devices
   - Launches VersaMail and adds a new email with an attachment
