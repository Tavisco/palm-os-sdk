---------------------------------------------------------------------------
Name

  FileBrowserSample

---------------------------------------------------------------------------
Description

  This sample code shows how to use File Browser library and APIs.
  You should run this sample code on LifeDrive and Tungsten T5 devices.

  Please refer to the Palm Developer Guide for more inforation about the 
  File Browser library.

---------------------------------------------------------------------------
Builds With

  CodeWarrior
  Gcc

---------------------------------------------------------------------------
Devices

  LifeDrive 
  Tungsten T5

---------------------------------------------------------------------------
Requirements


---------------------------------------------------------------------------
Libraries Used

  File Browser Library

---------------------------------------------------------------------------
How to Run

  1. Place a text file onto an SD card
  2. Insert the SD card into the device
  3. Launch the application
  4. Select the "Card" symbol on the top right corner of the screen
  5. Tap and hold on the test file that you placed onto the SD card
  6. On the menu, choose "Open with..." by tapping on it
  7. Select "Sample" from the list under "select application"
  8. Tap "OK"
