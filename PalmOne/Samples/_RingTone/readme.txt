---------------------------------------------------------------------------
Name

  _RingTone

---------------------------------------------------------------------------
Description

  The _RingTone application is a sample application showing the use of 
  the shared Ring Tone Library.  It demonstrates how to get the list of 
  tones from the system database and how to start and stop playing a tone.

---------------------------------------------------------------------------
Builds With

  CodeWarrior
  Gcc

---------------------------------------------------------------------------
Devices

  All Treo smartphones

---------------------------------------------------------------------------
Requirements


---------------------------------------------------------------------------
Libraries Used

  Ring Tone Library

---------------------------------------------------------------------------
How to Run

  1. Launch the 'Tone' application
  2. Make sure ringer is in 'ON' position
  3. Tap on 'prev' or 'next' to select ring tone
  4. Tap on 'play'
   - a. The ring tone should play
     b. Tapping on the screen while tone is playing should stop playback
